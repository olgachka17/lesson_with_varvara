# Папка для дипломов и сертификатов

Загрузите сюда ваши файлы дипломов в формате JPG, PNG или PDF.

Рекомендуемые названия файлов:
- diploma-bachelor.jpg (или .png, .pdf) - Диплом бакалавра
- certificate-c1.jpg - Сертификат C1
- diploma-aspirantura.jpg - Аспирантура
- certificate-scotland.jpg - Сертификат из Шотландии

Можете добавить дополнительные сертификаты с любыми названиями.
